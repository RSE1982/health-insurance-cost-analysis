Healthcare Insurance Cost Analysis

This project analyzes healthcare insurance costs based on various factors such as age, BMI, smoking status, and region.

Features

  • Data Cleaning and Preprocessing
  • Exploratory Data Analysis (EDA)
  • Statistical Analysis
  • Predictive Modeling
  • Visualization of Results

Technologies Used

  • Python
  • Pandas
  • NumPy
  • Matplotlib
  • Seaborn
  • Plotly

Getting Started

To get started with this project, clone the repository and install the required dependencies listed in the requirements.txt file.

Interactive Visualisations

  • Categorical Feature Distribution
  • Charges by Smoking Status
  • Charges by Gender
  • Charges by BMI
  • Charges by Age
